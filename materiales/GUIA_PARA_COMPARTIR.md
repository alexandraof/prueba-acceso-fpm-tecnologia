## 🚀 Opciones para Compartir con tus Alumnos

### Opción 1: Google Classroom (Recomendado) ⭐
1. Abre Google Classroom
2. Crea una nueva tarea o anuncio
3. Adjunta el archivo `Materiales_Estudio_Grado_Medio.zip`
4. Agrega instrucciones: "Descarga y descomprime el archivo. Abre los archivos .md con cualquier editor de texto o navegador web"
5. **Ventaja**: Los alumnos pueden descargar directamente desde la plataforma

### Opción 2: Google Drive / OneDrive
1. Sube el archivo ZIP a tu Google Drive o OneDrive
2. Genera un enlace de compartir
3. Comparte el enlace con tus alumnos por email o WhatsApp
4. **Ventaja**: Fácil de actualizar si necesitas modificar algo

### Opción 3: Email Masivo
1. Crea un email con el archivo adjunto
2. Usa el campo BCC para proteger la privacidad de emails
3. Asunto: "📚 Materiales de Estudio - Preparación Examen Grado Medio"
4. **Limitación**: Tamaño máximo de adjuntos (generalmente 25MB)

### Opción 4: Plataforma Educativa (Moodle, etc.)
1. Si tu centro usa una plataforma educativa
2. Sube el archivo como recurso de la asignatura
3. **Ventaja**: Integración perfecta con el entorno educativo

### Opción 5: Repositorio GitHub (Para alumnos avanzados)
1. Crea un repositorio público en GitHub
2. Sube todos los archivos
3. Comparte el enlace del repositorio
4. **Ventaja**: Versionado, actualizaciones fáciles, alumnos pueden contribuir
