## 🔧 **SOPORTE TÉCNICO**

### **Problemas Comunes y Soluciones**

#### **"No se ven las imágenes"**
- **Causa:** Archivos .md abiertos individualmente
- **Solución:** Usar siempre `index.html` o abrir desde `materiales/`

#### **"Los archivos .md se ven raros"**
- **Causa:** Editor de texto básico
- **Solución:** Arrástralos a un navegador web moderno

#### **"No puedo abrir index.html"**
- **Causa:** Navegador desactualizado
- **Solución:** Usar Chrome, Firefox, Edge o Safari moderno

### **Verificación Pre-Entrega**
Antes de entregar a los alumnos, ejecuta `verificar_entrega.bat` para confirmar que todo funciona.
