# 📚 Materiales de Estudio - Grado Medio Tecnológico

## 🎯 Información General

Estos materiales han sido creados específicamente para preparar a los alumnos para el **examen de acceso a grado medio en la rama tecnológica**. Incluyen teoría completa, ejercicios tipo test similares al examen real, y visualizaciones profesionales de fórmulas y diagramas.

### 📊 Estadísticas del Contenido
- **210 ejercicios** tipo test con soluciones detalladas
- **36 imágenes** profesionales (diagramas, fórmulas visuales)
- **3 temas completos**: Dibujo Técnico, Máquinas y Mecanismos, Electricidad
- **100% autónomo**: Los alumnos pueden estudiar sin supervisión constante

## 📖 Cómo Usar los Materiales

### 1. **Acceso a los Materiales**
- **Opción A:** Abrir `index.html` en cualquier navegador web
- **Opción B:** Descargar archivos individuales desde la página web
- **Opción C:** Acceder directamente a la carpeta `materiales/`

### 2. **Lectura de Archivos**
- Los archivos `.md` se pueden abrir con:
  - **Navegador web** (Chrome, Firefox, Edge, Safari)
  - **Editor de texto** (Bloc de notas, VS Code, Sublime Text)
  - **Aplicaciones de Markdown** (Typora, Mark Text)

### 3. **Método de Estudio Recomendado**

#### **Semanas 1-2: Fundamentos**
- Estudiar la teoría de cada tema
- Practicar ejercicios básicos (primeros 20-30 de cada tema)
- Memorizar fórmulas usando las imágenes de referencia

#### **Semana 3: Aplicación**
- Resolver ejercicios avanzados y mixtos
- Combinar conceptos de diferentes temas
- Practicar resolución de problemas complejos

#### **Semana 4: Repaso Final**
- Simulacros completos del examen
- Revisar errores y conceptos débiles
- Repaso intensivo de fórmulas y procedimientos

## 🔧 Solución de Problemas

### "No se ven las imágenes"
- Asegúrate de que las carpetas `imagenes/` estén en la misma ubicación que los archivos `.md`
- Si abres archivos `.md` individualmente, las rutas de imagen pueden no funcionar
- **Solución:** Usa siempre la página `index.html` o abre los archivos desde la carpeta `materiales/`

### "Los archivos .md se ven como texto plano"
- Los archivos Markdown necesitan un visor compatible
- **Solución:** Arrástralos a un navegador web moderno

### "No puedo descargar desde la página web"
- La página funciona sin conexión a internet
- **Solución:** Abre `index.html` directamente desde la carpeta

## 📞 Soporte

Si encuentras problemas con los materiales:
1. Verifica que todos los archivos estén en sus carpetas correctas
2. Asegúrate de usar un navegador web moderno
3. Consulta la guía completa en `GUIA_PARA_COMPARTIR.md`
